# 2. Основные функции

## 2.1 Вход

Пользователь должен иметь возможность зарегистрироваться и войти в систему.  
Авторизация реализуется через сторонние OAuth-провайдеры.

## 2.2 Управление целями

Пользователь может создавать **цель**.  
Пользователь может создавать промежуточные шаги у **цели**.

Пользователь может редактировать или отменять цель.  
Отменённая цель считается закрытой с вычетом звёзд.

## 2.3 Обработка целей

За выполнение цели пользователь получает звёзды.  
За отмену цели — штраф: списание звёзд.
Если звезд 0, то тогда баланс звезд уходит в минус.

## 2.4 Просмотр и завершение шагов

Пользователь может просматривать шаги на цели.
На не запущенных целях он может удалять шаги и редактировать дедлайны.
На запущенных целях он может только завершить шаг.

## 2.5 Просмотр целей

Пользователь может просматривать список целей других пользователей.

В списке отображаются:

- Статистика по количеству достигнутых целей и счет звёзд
- Возможность создать **награду** для завершенной цели, если не превышен лимит по просроченным шагам
- Возможность создать большую награду для другого пользователя, если у него есть нужное кол-во звезд

## 2.6 Создание награды

Пользователь может создать награду:

- Для закрытой цели другого пользователя, если она будет закрыта без 20% (далее будет детализировано) штрафов
- За определённое количество полученных звёзд за закрытые цели

## 2.7 Статистика и награды

За выполнение каждой цели пользователь получает звёзды (например, +10 звёзд).

В профиле отображается:

- Общее число достигнутых целей
- Общее число полученных звёзд
