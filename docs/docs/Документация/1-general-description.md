# 1. Общее описание

Создать веб-приложение, в котором пользователи могут:

- создавать цели
- прикреплять промежуточные шаги к целям
- завершать цели и шаги
- просматривать цели других пользователей
- прикреплять награды к завершенным целям, если шаги закрыты без нарушений сроков
- прикреплять награды к пользователям, у которых достаточно звёзд
- видеть статистику по звездам и штрафам
- фильтровать цели по статусу, дате, автору
