# 1. Общее описание

Создать веб-приложение, в котором пользователи могут:

- создавать цели
- прикреплять промежуточные шаги к целям
- просматривать цели других пользователей
- прикреплять награды к завершенным целям
- прикреплять награды к пользователям, у которых достаточно звёзд

Другие пользователи также могут видеть статистику по звездам и штрафам.
