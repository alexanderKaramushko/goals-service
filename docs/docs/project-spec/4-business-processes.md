# 4. Логика бизнес-процессов

**Создание цели**:

1. пользователь заходит в систему
2. создает цель, указывает время, когда нужно ее завершить
3. добавляет шаги, указывает время завершения для каждого шага
4. нажимает "Старт"

**Назначение награды к цели**:

1. другой пользователь заходит в систему
2. добавляет награду к завершенной цели, если цель закрыта без превышения лимитов по просроченным дедлайнам

**Назначение награды за звезды**:

1. другой пользователь заходит в систему
2. переходит в профиль пользователя
3. создает награду за звёзды
4. у другого пользователя списываются звёзды, если он принимает награду

**Отмена цели**:

1. пользователь заходит в систему
2. переходит к цели и нажимает кнопку "Отменить"
3. фиксируется дата отмены
4. списываются звёзды за отмену

**Выполнение шага**:

1. пользователь заходит в систему
2. переходит к цели
3. переходит к шагу у цели
4. завершает шаг, описывает итоги шага

**Получение награды**:

1. пользователь заходит в систему
2. переходит к завершенной цели
3. выбирает "Получить награду"
